# 2aac230: feat(export): eksport CSV danych snu z Historii (Faza 3)

**Data:** 2026-06-25
**Branch:** main
**Faza zadania:** Faza 3 — Eksport / raport dla pediatry (część CSV; PDF/Edge Function = osobna iteracja)

## Co zostalo zrobione
- Eksport historii snu do pliku CSV z poziomu ekranu Historia. Ikona (Upload) w
  nagłówku → bottom-sheet z wyborem zakresu (presety 2 tyg / 30 dni / 90 dni) i
  przyciskiem „Eksportuj CSV”.
- Czysta serializacja `sessionsToCsv`: jeden wiersz per sesja, kolumny
  Data;Start;Koniec;Długość;Typ;Notatki;Tagi. Separator `;` (Excel PL), escaping
  RFC 4180 (cudzysłowy/separator/nowa linia), typ PL (Drzemka/Sen nocny), tagi
  przez `tagLabel`, sesja w toku → puste Koniec/Długość, sort chronologiczny, CRLF.
- Dostarczenie pliku: Web Share API (`navigator.canShare({files})` → arkusz
  udostępniania na iOS/Android) z fallbackiem na klasyczne pobranie (`<a download>`).
  BOM UTF-8 dla poprawnego renderu polskich znaków w Excelu. `AbortError`
  (anulowanie share) traktowane jako `cancelled` (bez komunikatu błędu).
- Wydzielenie `fetchSessionsInRange` z `useSessions` queryFn — single source dla
  listy Historii i eksportu (eksport to akcja jednorazowa, poza cache TanStack).
- Feedback wyniku przez istniejący `useSnackbar` (wyeksportowano / brak sesji / błąd).
