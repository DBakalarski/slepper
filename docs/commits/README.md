# Commit logs

Per-commit dokumentacja. Jeden plik = jeden commit. Format opisany w `sleeper-app/CLAUDE.md` (sekcja "Commit logging").

**Konwencja nazewnictwa:** `YYYY-MM-DD-<short-hash>-<slug>.md`

**Sortowanie:** chronologiczne po dacie (najnowsze na dole listy `ls`).
